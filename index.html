Conoce a Jaime Sifontes

Desde muy chico he sentido una inmensa pasion por la tecnologia, motivado y siempre en busca de nuevos retos, con diversas habilidades que me han permitido desarrollar y diseñar sitios web con los mas altos estándares de calidad. Cuento con experiancia en otras areas como Gestion de proyectos, he participado tanto como miembro de un equipo de trabajo, como lider de proyectos. Me dedico a mantenerme al día con las últimas tendencias y tecnologías de la industria. Soy una persona orientada al equipo con una actitud positiva y una pasión por crear sitios web y aplicaciones web innovadoras y fáciles de usar.

Experienca

Diseño UX/UI

Experiencia en varios aspectos del diseño de UX, incluida la investigación de usuarios, arquitectura de la información, wireframing, prototipos, pruebas de usabilidad y el pensamiento de diseño. Creación de diseños centrados en el usuario que cumplen objetivos comerciales y necesidades del usuario utilizando herramientas de diseño.

Figma AdobeXD Miro Sketch

Desarrollo Web

Experiencia en desarrollo web front-end, utilizando diferentes lenguajes marcado, estilo y de programación como HTML, CSS y JavaScript también usa marcos y bibliotecas como Bootstrap y React, y diseño de bases de datos. Comprensión del diseño responsivo "mobile first", accesibilidad web y mejores prácticas de SEO.

HTML CSS Bootstrap JavaScript React

Project Management

Conocimientos y habilidades prácticas en varios aspectos del proyecto gestión incluyendo la planificación, ejecución, seguimiento, riesgo y gestión de las partes interesadas, la comunicación y la familiaridad con metodologías como Agile, Waterfall e Hybrid.

Waterfall Agile Scrum Methodology Assana Jira MS Project Microsoft Office
